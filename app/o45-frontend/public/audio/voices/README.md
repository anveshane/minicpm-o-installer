# 语音音频文件目录

这个目录用于存放语音选择器组件中使用的音频文件。

## 文件命名规范

建议使用以下命名规范：
- `warm-female.mp3` - 暖音（温柔而治愈的女声）
- `deep-male.mp3` - 沉稳（低沉磁性的男声）
- `bright-male.mp3` - 晨煦（阳光、温暖、精力充沛的大男孩）
- `child-voice.mp3` - 童趣（俏皮可爱的童声）
- `elegant-female.mp3` - 优雅（成熟稳重的女声）
- `energetic-young.mp3` - 活力（充满活力的年轻声音）
- `profound-male.mp3` - 深沉（深沉有力的男声）
- `fresh-girl.mp3` - 清新（清新自然的少女音）

## 音频文件要求

1. **格式**：建议使用 MP3 格式，兼容性好
2. **时长**：建议 3-10 秒，展示语音特点
3. **质量**：建议 128kbps 以上，确保音质清晰
4. **内容**：建议包含该语音特点的简短对话或介绍

## 使用方法

音频文件会通过 `/audio/voices/` 路径访问，例如：
```javascript
url: '/audio/voices/warm-female.mp3'
```

## 注意事项

- 确保音频文件大小适中，避免影响页面加载速度
- 音频文件应该是该语音的典型代表，能够体现语音特点
- 建议使用相同的音频格式和编码参数，确保一致性
